# Documents — Atsushi Yamagishi

This repository stores PDFs (papers, slides, and drafts) by Atsushi Yamagishi. It's a document archive, not a software project — there is no build or test system.

Development / contribution
- To view files: open the PDFs in this folder with your PDF viewer.
- To contribute: add new PDF or source files, commit on a feature branch, and open a pull request to merge into main.
- No automated tests or CI detected in this repository.

Repository maintenance
- Keep filenames descriptive and include dates or version info when relevant.
- If adding source (LaTeX, R, etc.), include a short README in that subfolder describing how to build the PDF.

Contact
- For questions, open an issue or contact the repository owner.
